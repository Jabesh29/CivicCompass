# CivicCompass

🚀 CivicCompass 

Intelligent Government Scheme Eligibility Engine

📌 Overview

CivicCompass AI is a cloud-based eligibility engine that helps citizens discover government welfare schemes they qualify for.
By entering basic demographic and economic details, users receive:
✅ Personalized scheme recommendations

✅ Clear eligibility explanations

✅ Multilingual support

✅ Fast and scalable cloud-based processing


🎯 Problem Statement

Millions of citizens struggle to identify which government schemes they are eligible for due to:

Complex eligibility rules
Scattered information sources
Language barriers
Lack of digital guidance


Our solution simplifies access using an intelligent cloud-based system.

💡 Solution

CivicCompass AI allows users to:

1. Enter personal details (age, income, category, occupation)
2. Instantly receive eligible schemes
3. View clear reasoning for eligibility
4. Access information in preferred language


☁️ AWS Services Used

Amazon S3 – Static website hosting
Amazon API Gateway – REST API endpoint
AWS Lambda – Backend eligibility processing
Amazon DynamoDB – Scheme data storage
Amazon Translate – Multilingual support


🏗 Architecture Flow

User → S3 Web App → API Gateway → Lambda → DynamoDB → Translate → Response

🛠 Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Python (AWS Lambda)

Database: DynamoDB

Cloud Platform: AWS (Serverless Architecture)
